## F. デザインの雰囲気

### ベースデザイン
**awesome-design-md の Mastercard をベース** + ミニマル洗練系のカスタムパレット

- DESIGN.md URL: https://getdesign.md/mastercard/design-md
- 既存サイトのクリーム＆テラコッタ路線は **「洗練・ミニマル方向にリブランド」**

### 確定カラーパレット 🌅「マジックアワー」テーマ

夕焼けが終わって星が出る直前、空が「ピーチ→ピンク→ラベンダー→スカイブルー」の幻想的な4色グラデになる時間帯＝マジックアワーをモチーフ。  
**「新しい人生・解放・希望・幻想美」** を象徴。

| 役割 | カラー | コード | 使い所 |
|---|---|---|---|
| **背景** | 純白 | `#FFFFFF` | サイト全体のベース、パキッと潔く白 |
| **太陽（ピーチ）** | サンセットピーチ | `#FFB897` | グラデ起点、暖かみの主役 |
| **コーラル（ピンク）** | ホットコーラル | `#EE9C9C` | グラデ第2層、女性らしさ |
| **ラベンダー（薄紫）** | マジックラベンダー | `#C5A2D4` | グラデ第3層、幻想・マジックアワーの核 |
| **海・空（水色）** | ソフトスカイブルー | `#7DB3D0` | グラデ終点、解放・希望 |
| **CTA・強調色** | ディープコーラル | `#D87C5E` | ボタン・リンク（読みやすさ優先で単色） |
| **本文テキスト** | チャコール | `#2C2C2A` | 本文（完全な黒ではなく温かみのある墨色） |
| **サブテキスト** | ウォームグレー | `#5F5E5A` | キャプション、補足文 |

### グラデーションの使い方

**ヒーローの装飾円・セクション背景の装飾**：4色の連続グラデーション
```
linear-gradient(160deg, #FFB897 0%, #EE9C9C 30%, #C5A2D4 65%, #7DB3D0 100%)
```

**見出しのキーワード強調テキスト**：3色のドラマチックグラデ
```
linear-gradient(120deg, #D87C5E, #B86A8C, #5A8EAA)
```
（例：「**依存しない**」「**新しい**」など人生のキーワードに）

### デザインの要点
- 白背景 × マジックアワーのグラデ装飾円が中心
- グラデは「装飾円」「見出しのキーワード」のみに使用（多用しすぎない）
- それ以外は単色で読みやすさ最優先
- 4色の中で **ラベンダー** が他のサイトにない差別化ポイント

### デザイン特徴（採用する要素）
| 要素 | 採用理由 |
|---|---|
| **純白背景** | ミニマル・モダン・洗練を最大化、テラコッタが映える |
| **重なる丸い形（淡テラコッタ × ダスティローズ）** | やわらかい女性らしさ、Mastercardの円モチーフを継承 |
| **大胆な大型ディスプレイタイポ** | 「夫にも会社にも依存しない働き方」が映える |
| **丸枠での写真演出** | 既存の丸い顔写真と統一感 |
| **明朝フォント（Zen Old Mincho）の見出し** | 上品さ、エッセイ感 |
| **雑誌・編集記事風レイアウト** | 物語の3章立てが活きる |
| **広めの余白** | 「大人の余白」、洗練感の核 |

### 全体トーン
- ミニマル × 洗練 × 大人の女性ブランド
- 「雑誌のエッセイ記事を読む」感覚
- 50代女性が見ても「ちゃんとした人」と思える格
